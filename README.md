# 📊 Retail Sales & Product Performance Dashboard | Power BI

## 📌 Project Overview
Developed an interactive **Retail Sales & Product Performance Dashboard** in Power BI to analyze revenue, orders, profit, returns, and product/customer insights.  
This project demonstrates skills in **data modeling, DAX, Power Query transformations, and business intelligence storytelling**.

---

## 📂 Repository Contents
- **Datasets/** → Raw Excel datasets (customer, product, territory, returns, calendar)  
- **Sales Data/** → Transactional sales records  
- **Retail Sales And Product Performance Dashboard.pbix** → Power BI dashboard file  
- **README.md** → Documentation  
- **LICENSE** → MIT License  

---

## 🚀 Key Features
- Executive dashboard with KPIs: revenue, orders, profit, returns  
- Product performance analysis (top products, categories, subcategories)  
- Customer segmentation by occupation and income level  
- Geographic insights with interactive map visualizations  
- Decomposition tree and key influencer analysis for deeper insights
